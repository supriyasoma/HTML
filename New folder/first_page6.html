First HTML PAGE

Create an HTML file (e.g. first_page.html) that specifies a page that contains a heading and two paragraphs of text. Use the HTML tags <h1> and <p> in this exercise. As the texts in the heading and paragraphs you can use any texts you like

The <h1> to <h6> tags are used to define HTML headings <h1> defines the most important heading. <h6> defines the least important heading
